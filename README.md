귀몽이의 블로그
---
이 블로그는 지식 정리 및 공유를 위한 개인 블로그입니다.  
해당 블로그는 Jekyll을 이용하여 작성되었습니다.

# 설치

[[상세 가이드]](https://jekyllrb.com/docs/installation)

# 실행

```shell
$ jekyll serve
```
